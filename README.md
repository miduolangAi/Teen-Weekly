---

#### 🧐 关于这份周报

欢迎来到《少年新知周报》的开源仓库！

这不是那种让你头大的严肃读物，而是一份**由少年主导、为少年打造**的极客周刊。我们的定位很明确：

- **拒绝说教**：用“局长”的视角看世界，像聊八卦一样聊科技。
- **硬核整活**：把实验室搬进厨房，把物理公式藏进游戏里。
- **万物皆可测**：从奶茶甜度到地球曲率，没有什么是不能研究的！

---

#### 📂 仓库导航

为了让你不迷路，这里是我们的“藏宝图”：

- `/issues`：**周报编辑部**。每一期周报的草稿、定稿都在这里；也是你的投稿箱。
- `/video-scripts`（预留）：**短视频剧本**。存放我们在抖音/B站发布的“科学快闪”视频脚本。
- `/experiments`（预留）：**翻车档案馆**。记录那些“炸厨房”的实验过程和反思。

---

#### 📅 更新计划

- **每周日晚**：发布最新一期周报（在 Issues 中查看）。
- **不定期**：掉落“局长挑战”任务和实体小礼物。

---

#### 🤝 参与共建

我们采用开源共建模式，欢迎所有“见习探员”加入：

1. **阅读周报**：前往 `Issues` 查看最新一期。
2. **提交脑洞**：点击 `New Issue`，选择模板，提交你的想法。
3. **捉虫**：发现错别字或逻辑漏洞？欢迎提交 `Pull Request`。
