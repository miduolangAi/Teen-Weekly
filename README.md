## 📝 关于本项目

这是一个专门为中学生打造的开源周报项目。我们的目标是：
1. **🌐 科技前沿**：每周精选 10 条科技新闻，拒绝信息差。
2. **🎮 游戏观察**：不仅仅是玩，更是从设计和文化角度观察游戏。
3. **📚 学习重点**：记录各科难点，分享一道经典错题。
4. **🎬 周末推荐**：好书、好电影推荐，丰富课余生活。
